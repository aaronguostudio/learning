# HTTP

## Concepts

- URI and URL
  - URI contains URL and URN
    - URL 像一个地址，URN 像一个名字（唯一标识）
    - 能称之为 URL，都要有一个访问方式（机制），比如
      - ftp://...
      - http://...
      - telnet://...
    - URI, 比如
      - www.imooc.com // 这是一个 URI 因为没有指定：HTTP 还是其他访问机制
