# Data Structure

## Real life examples

- Database
  - tree: AVL, 红黑树，Treap, 伸展树，B 树
  - hashtable
- 操作系统
  - stack
    - 系统栈，函数的循环调用
  - heap
    - 优先队列
- 文件压缩
  - 哈弗曼树(比较简单的算法)
- 通讯录
  - trie - 前缀树
- 游戏
  - 寻路算法
    - DFS 深度优先
    - BFS 广度优先
